Bootstrap Strength Meter

Bootstrap Strength Meter is a simple password strength meter based on Password Score for strength estimation. The plugin uses different techniques to visualize the strength of the password which is measured in the means of entropy. You can choose between different visualization options:

  • Progress Bar
  • Text
  • Tooltip

Use the example below to find out the strength of your passwords! — See the explanation below for how to use the plugin.

  • Usage
  • Code

Bootstrap Strength Meter is based on Password Score. Therefore, we first need to include Twitter Bootstrap, jQuery and Password Score: 

<!-- Bootstrap CSS -->
<link rel="stylesheet" type="text/css" href="docs/css/bootstrap.css">

<!-- jQuery, Password Score (keyboard.js and score.js) and Bootstrap Strength Meter -->
<script type="text/javascript" src="docs/js/jquery.js"></script>
<script type="text/javascript" src="password-score/js/keyboard.js"></script>
<script type="text/javascript" src="password-score/js/score.js"></script>
<script type="text/javascript" src="dist/js/bootstrap-strength-meter.js"></script>

Password Score may use several dictionaries to give a realistic estimate of the strength of a password, for example german and english dictionaries as well as dictionaries of common first and last names. Some dictionaries are provided by Password Score and need to be loaded via AJAX. In addition, Password Score may search for keyboard patterns. Therefore, both a QWERTY as well as a QWERTZ keyboard is provided. The plugin expects all dictionaries and keyboards to use as parameters. In the below example, #example-container refers to the container the text is added to: 

// Password Score uses different kinds of dictionaries to estimate
// the strength of a password: english, german, cities, countries,
// common last and first names, common passwords.
// All these dictionaries are available as JSON and are loaded:
$.when(
        $.getJSON('password-score/data/json/en.json'), // 'dictEN' - english dictionary.
        $.getJSON('password-score/data/json/de.json'), // 'dictDE' - german dictionary.
        $.getJSON('password-score/data/json/cities.json'), // 'cities' - city names.
        $.getJSON('password-score/data/json/countries-en.json'), // 'countriesEN' - country names english.
        $.getJSON('password-score/data/json/countries-de.json'), // 'countriesDE' - country names german
        $.getJSON('password-score/data/json/female.json'), // 'female' - female first names.
        $.getJSON('password-score/data/json/last.json'), // 'last' - last names.
        $.getJSON('password-score/data/json/male.json'), // 'male' - male first names.
        $.getJSON('password-score/data/json/passwords.json') // 'passwords' - common passwords.
).then(function(dictEN, dictDE, cities, countriesEN, countriesDE, female, last, male, passwords) {

    // The dictionaries to use:
    var dictionaries = [
        dictEN[0],
        dictDE[0],
        last[0],
        male[0],
        female[0],
        cities[0],
        passwords[0],
        countriesEN[0],
        countriesDE[0]
    ];

    // Password Score may search for common keyboard patterns.
    // The library provides both a QWERTY and a QWERTZ keyboard.
    // Use both keyboards for estimation:
    var keyboards = [
        QWERTY,
        QWERTZ
    ];

    // Initialize the plugin to use text and use
    // the above dictionaries and keyboards.
    $('#example').strengthMeter('text', {
        container: $('#example-container'),
        dictionaries: dictionaries,
        keyboards: keyboards
    });
});
